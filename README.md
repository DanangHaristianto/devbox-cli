# 🚀 DevBox CLI

DevBox CLI adalah tool automation untuk Termux.

# ⚡ Fitur
- Install tools otomatis
- Backup ke GitHub
- Deploy cepat
- Update dari repo

# 🛠️ Cara Pakai
devbox install
devbox backup
devbox deploy
devbox update

# 💡 Tujuan
Mempermudah developer di Termux agar tidak setup manual dan bisa kerja lebih cepat.
